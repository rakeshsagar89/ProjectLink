# React + Spring Boot - Link Shortener

This project is a full-stack combining **Spring Boot** for the backend and **React** for the frontend. It provides a working setup for building modern web applications using Java and JavaScript.

---

## Prerequisites
- ✅ Java 8 or higher (JDK)
- ✅ Node.js (v14 or higher recommended)
- ✅ npm (comes with Node.js)
- ✅ Git (for cloning the repo)

---

Spring-boot code is inside server directory you can start the app using
./mvnw spring-boot:run      # macOS/Linux
mvnw spring-boot:run        # for Windows
server will run at http://localhost:7070/


Frontend Setup (React)
React JS code is inside client directory you can start the app using

Install all necessary React dependencies:

npm install       # Install dependencies
npm start         # Start dev server
npm run build     # Create production-ready build

This will start the React app at:
http://localhost:3000/
